parsed_all_data_raw.csv - без изменений. Не все редакторы читают, питон подхватывает.
parsed_all_data_cleaned.csv - убраны везде двойные кавычки. Изменены данные, но офис теперь открывает без ошибок.
parsed_all_data_quoted.csv - каждая ячейка экранирована символом |. Должно работать везде, но файл перегружен.
parsed_all_data.json - собраные данные, чтобы удобнее работать.
